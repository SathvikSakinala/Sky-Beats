SkyBeats

Our Mini Project is to create a music player like web application where a single user or a group of users can listen to songs. Here, each user can create a profile and make friends. Users can also upload their own songs and other users will be able to hear them. Users can even like and comment on songs. So, this is like a social media platform especially for music lovers. The technologies we are going to use are MongoDB for storing the songs and all other information. The web application will be developed using HTML, CSS, JavaScript and Django.
